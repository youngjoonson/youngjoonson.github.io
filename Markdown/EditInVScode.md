# Markdown and Visual Studio Code

## Markdown extensions

1. markdownlint
2. Markdown Theme Kit
3. Markdown TOC
4. Markdown Shortcuts

## Markdown preview

- shift command V : switch between views, editing, preview
- command K and then V : open the preview to the side

## Dynamic previews and preview locking

- By default, preview automatically update to preview the currently active markdown file
- more actions -> toggle preview locking

## Editor and preview synchronization

- By default, VS Code automaticlly synchronize the editor and the preview pane
- shift command P -> open settings (JSON)
  - markdown.preview.scrollPreviewWithEditor
  - markdown.preview.scrollEditorWithPreview
- selected line in editor -> a light grey bar in the left margin in the preview
- double clicking an element in the Markdown preview will open the editor for the file and scroll to the line nearest the clicked element.

## Outline view

- show the symbol tree of the currently active editor, the Markdown file's header hierarchy.
